# TrackWise LDRC

An elderly wellbeing monitoring system for the Learning & Development Resource Centre (LDRC). It aggregates activity signals from an Android phone, an Alexa device, and a CCTV camera into a Google Sheets dashboard, firing Telegram and WhatsApp alerts when silence thresholds are breached.
